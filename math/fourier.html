Let us recall the basic point-wise convergence criterion (theorem 2.1).
Theorem 2.2: Pointwise Fourier convergence
If a function f is nice enough (piecewise-smooth and 2π-periodic) then its
Fourier series converges point wise to f .
Piece-wise continuous is a milder condition than continuity. If f is piecewise-
continuous and you pick any interval I, then f only has ﬁnitely many discontinu-
ities in I and these discontinuities have all be jump discontinuities (and perhaps
also removable discontinuities). We don’t want crazy oscillation like sin 1/x or
blowing up to inﬁnity like 1/x. We want to be able to take the integral (area
under the curve) and talk about inner product!
Another way to describe piecewise-continuous function f is to say that we
can partition the domain into open intervals, where f is continuous on each of
the intervals. So the only potential sources of discontinuities come from the
endpoints.
